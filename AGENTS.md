# AlgoMentor Project

Goal:
Build a DSA Visualizer & Execution Tutor.

Tech Stack:
- Frontend: React
- Backend: Node.js + Express
- Database: PostgreSQL (later)

Rules:
- Build in modules
- Keep code clean and modular
- Start simple (loops + arrays only)

Modules:
1. Code Input UI
2. Execution Engine
3. Visualization Engine

Commands:
- frontend: npm run dev
- backend: node server.js
